> rrd/apache_localhost.rrd I get the feeling that there is something wrong.

The size depends on how much fields are saved. All rows are created when
starting the file, so the filesize will not change unless you edit the
rrd-file (e.g. add/del fields). This is no problem, it is the expected
and required behavior for rrdtool.
