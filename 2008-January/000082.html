SELF SOLVED:

config/graphs.php

e.g.

$config['list'][] = array(
       'categories' => array('host' => 'tv'),
       'title' => 'TV',

and so on..

with 'categories' => array('host' => 'tv')," you define the assignment 
of each graph.

now you have to add the menu entry:

$config['tree'] = array(
        'title' => 'All graphs',
        'filter' => '',
        'sub' => array(
                array(
                        'title' => 'localhost',
                        'filter' => 'host:localhost'
                ),
                array(
                        'title' => 'SUPER',
                        'filter' => 'host:super'
                ),
and so on...

Have fun!
