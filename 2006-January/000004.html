> I need to graph informations from  my  postfix MTA. I use clamav and
> spamassasin. I've installed mailgraph tool.
> I would like to know how can I use specific rrd file ? I don't want to
> create just read datasource and create associated graphs.
> 
> I've the same need for courier "imap"  and "pop3".

You can use rrdfiles created by external scripts using the following config:
    [...]
    array(
      'type' => 'YOUR_TYPE',
      'file' => '/path/to/file.rrd', // thats the important part
      'name' => 'your_name', // name the datasource
      'ds' => 'your_dsname',
      'cf' => 'YOUR_CF'
    ),
    [...]
You may use the optional additional values like "legend" or "color", too.
The parameter "name" is needed by rrdtool, it is called "vname" there:
DEF:vname=rrdfile:ds-name:CF[:step=step][:start=time][:end=time]
(from 'man rrdgraph')

Note: This feature is not tested very well. If you find any errors,
please report them.
