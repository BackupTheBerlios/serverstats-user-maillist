Seems like this Optimizer handles include()'s somehow different than the
normal PHP does. This might be a big problem for using Serverstats.

I have two different workarounds in mind:
 * Comment the simpleconfig-lines, this is the only thing using
$rootConfig (but simpleconfig will not work after this)
 * Put $rootConfig = $GLOBALS['config']; before the simpleconfig-lines

If one of these (or both) helps, please tell me. I want to put it into
the FAQ, to have a solution available for anyone.
