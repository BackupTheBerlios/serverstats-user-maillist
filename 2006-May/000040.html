> I plan to use serverstats on vserver host (linux-vserver.org)

There is no support for SNMP in Serverstats. It is planned, but I have
no SNMP-server to test anything I write.

> If yes how can
> I setup it ?

You can write your own SNMP-class. If you do so I would be interested in
getting a copy.
(See the included sources for examples)

> Can I use simple config and the normal config on the same time  ?
> I've got  0.7 config for postfix so  I would like to reuse it

You can disable SimpleConfig if you have a working configuration
already. If you want to use SimpleConfig you can use your own
configuration at the same time. SimpleConfig only write to the normal
config, but uses something like template to make it easier to use.
